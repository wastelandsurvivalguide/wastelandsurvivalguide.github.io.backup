Final Steps

  1. Download the WSG load order here.
  2. Once the download has finished, click the button at the top of MO2 and select Open Profile folder.
  3. Extract the loadorder.txt from the archive into the opened folder and overwrite when prompted.
  4. Once the file is extracted, press F5 in MO2.
  5. If everything went correctly, the final plugin in MO2's right pane should be DN Weathers.esp.
  6. Click the Backup button so if something ever goes wrong you can instantly restore this load order:
