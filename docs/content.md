# [**Wasteland Survival Guide**](./index.html)
## **Content**

### **First, create a "Content" [separator](./img/mo2 separator.png) in MO2.**
##### **As you install the mods from this section, organize them underneath this separator.**

|[AWOP4FO3](https://www.nexusmods.com/newvegas/mods/66265)<br>TTW A World of Pain for Fallout 3|Adds many intense and unique locations to the Capital Wasteland, new items, 4D storage, two companions and a couple of quests.|<p>- **Main File - AWOP4FO3**</p><p>- **Optional File - JIP CCC Avatars**</p><p>- **Optional File - [TTW AWOP4FO3 - No Starter Terminal](./dl/TTW AWOP4FO3 - No Starter Terminal.7z)** </p><p>Wasteland Starting Gear already provides starting loadouts and the 4D storage is overpowered; that is why the terminal is suggested to be disabled.</p><p>- **Patch File - [AWOP FO3 - TTW Patch](https://www.nexusmods.com/newvegas/mods/77945)**</p>|
| :- | :- | :- |
|[Casino Crowds](https://www.nexusmods.com/newvegas/mods/71037)|Increased NPC population in all 3 Casinos: 80+ across Gomorrah, 55+ across The Tops, 15+ in The Ultra-Luxe.|<p>**Main File - Casino Crowds**</p><p>**Patch File - [Casino Crowds - TTW Patch](https://www.nexusmods.com/newvegas/mods/77945)**</p>|
|[Functional Post Game Ending](https://www.nexusmods.com/newvegas/mods/66726)|Adds a fully functional post-game world that reflects the consequences of player choices & ending.|<p>- **Main File - Functional Post Game Ending**</p><p>- You will be prompted with a FOMOD installer; select only the **Tale of Two Wastelands Patch** then click Next and Install</p><p>- **Optional File - [FPGE - Casino Crowds Patch](./dl/FPGE - Casino Crowds Patch.7z)**</p>|
|[New Vegas Truce Series](https://www.nexusmods.com/newvegas/users/78355343?tab=user+files)|Allows for various main factions in the Mojave Wasteland to work together (or at least tolerate each other) when they previously couldn't.|Install the **Main File** for any "Truce" mods you plan to use on this playthrough.|
|[Rebuild the Capital](https://www.nexusmods.com/newvegas/mods/68030)|<p>- Expand the Brotherhood of Steel and their role in the wasteland following the events of Broken Steel.</p><p>- Under the player's guidance, the Brotherhood can establish relations with settlements and establish military outposts.</p><p>- The mod also features new armors, weapons, quests, perks, NPCs, and a recruitment system.</p>|<p>- **Main File - Rebuild the Capital**</p><p>&emsp;- In the FOMOD check **No Pony Express Boxes** (& **PAVE Patch** if you use Power Armor Visual Enhancement) then click Install</p><p>- **Patch File - [Rebuild the Capital - A World of Pain for Fallout 3 Patch](https://www.nexusmods.com/newvegas/mods/77945)**</p>|
|[The Living Desert](https://www.nexusmods.com/newvegas/mods/64623)|Adds several scripted events & hundreds of NPCs who will patrol roads, travel between towns, occupy locations, explore the desert, and react dynamically to your choices throughout the game.|<p>- **Main File - The Living Desert**</p><p>- **Patch File - [The Living Desert - TTW Patch](https://www.nexusmods.com/newvegas/mods/77945)**</p>|
|[The Someguy Series LE](https://www.nexusmods.com/newvegas/mods/48925/)|This master file allows you to play mods in the Someguy series, which are being remastered. There are currently three episodes available.|<p>- **Main File - Someguy Series 2 dot 0**</p><p>- For each episode, install the **Main File** and **Optional File - TTW Patch** if available:</p><p>&emsp;- [New Vegas Bounties I LE](https://www.nexusmods.com/newvegas/mods/77108)</p><p>&emsp;- [Bad Mothafucka LE](https://www.nexusmods.com/newvegas/mods/77229)</p><p>&emsp;- [New Vegas Killer LE](https://www.nexusmods.com/newvegas/mods/78427)</p>|
|[Transporters TTW](https://www.nexusmods.com/newvegas/mods/77645)|<p>Take on the Wastelands with a new alternative form of Fast-Travel, with over 37+ travel destinations! </p><p>It is strongly recommended to disable fast travel when using this mod, which it enables you to do easily via the MCM.</p>|<p>- **Main File - SD\_Transporters FOMOD**</p><p>&emsp;- Select **Tale of Two Wastelands** in the FOMOD installer</p><p>- **Patch File - [SD Transporters - AWOP FO3 Patch](https://www.nexusmods.com/newvegas/mods/77945)**</p>|
|[TTW Merged Collectable Quests](https://www.nexusmods.com/newvegas/mods/75829)|Adds quest markers for collectibles including: bobbleheads, steel ingots, alien captive recordings, intel briefcases & Keller family transcripts.|**Main File - TTW Merged Collectable Quests**|
|**TTW Quest Conversions**<br>by RoyBatterian|A series of Fallout 3 quests from various authors which add many new locations to the game, converted for TTW by RoyBatty|<p>**Main Files:** </p><p>- [TTW A Trail of Crumbs](https://www.nexusmods.com/newvegas/mods/66267)</p><p>- [TTW Another Interior Mod](https://www.nexusmods.com/newvegas/mods/66611)</p><p>- [TTW The Regulators](https://www.nexusmods.com/newvegas/mods/66266)</p>|
|**Uncut Wasteland & Extra Collection**|Restores a huge amount of scenery, NPCs, creatures and other details which were patched out New Vegas post-release. These versions are patched for TTW and FPGE by Kazopert.|<p>**Main Files:** </p><p>- [Uncut Wasteland](./dl/Uncut Wasteland.7z)</p><p>- [Uncut Extra Collection](./dl/Uncut Extra Collection.7z)</p><p>- **Patch File - [FPGE - Uncut Wasteland & Extra Collection Patch](./dl/FPGE - Uncut Wasteland & Extra Collection Patch.7z)**</p>|
|[Washington's Malevolence TTW](https://www.nexusmods.com/newvegas/mods/75027)|A huge quest mod where the Lone Wanderer travels to the fictional Maddox Island in search of the Rocky Cape Bank, a pre-War bank that is said to contain untold riches. Players will also get to explore multiple locations around the island with interesting backstories and meet plenty of different personalities that are sure to leave an impression that lasts.|- **Main File - Washington's Malevolence TTW**|
